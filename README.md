## Getting Started

### Install packages
```sh
npm install
```

### Install dependencies
```sh
npm install -g expo-cli
```

### Development
```sh
npm start
```

Clique em [http://localhost:3000/](http://localhost:3000/) para visualizar no browser.

Projeto [backend](https://github.com/Jhonatan-Pereira/node_omnistack_week_11).
